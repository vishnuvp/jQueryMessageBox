jQueryMessageBox

A jQuery plugin to create configurable status message boxes

Download .zip Download .tar.gz View on GitHub

A simple jQuery plugin to show up configurable CSS3 compliant messageboxes in a webpage.

Requires jQuery v1.4 or greater.

Usage: $.jQueryMessageBox(options);

options is an associative array of properties like type, behavior, title, text etc.
